This is a simple plain vanilla JS file hot-reload checker, It simulates browser hot-reload through a combination of polling and server-sde file checker. EXTREMELY SIMPLE SCript to deploy and use. Just add one line of code to the bottom of any web page that you want to re-load on file change. 

<script type="text/javascript" src="hotreload.js"></script>
The command above calls a PHP server file checker. hotreload.php
